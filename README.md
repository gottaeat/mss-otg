# mss-otg

## tree
```
├ misc/     -> miscellaneous build scripts.
└ utils/
  ├ bin/    -> assortment of scripts.
  ├ etc/    -> files complementing the scripts in ../bin/ and shells.
  └ share/  -> handlers for systemd and others.
```

## usage
```sh
usage: template.py [-h] [--prefix PREFIX] [--home HOME] [--setboxes]

OTG templater

options:
  -h, --help       show this help message and exit
  --prefix PREFIX  installation prefix
  --home HOME      $HOME of the user
  --setboxes       enable gottaeat/setboxes specific functionality
```
